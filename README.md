# Folder-Watching-Cat 🐱

特定のフォルダを監視して、命名規則に沿ったファイルやフォルダを検知したら通知してくれる常駐アプリケーションです。

## 機能

- 📁 特定のファイル名を検知して通知
- 📂 命名規則に一致するフォルダを検知して通知
- 🔮 （将来）検知時にスクリプトを自動実行

## 必要なもの

- Python 3.x
- watchdog（フォルダ監視ライブラリ）

## インストール

```bash
# 仮想環境の作成（推奨）
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# ライブラリのインストール
pip install watchdog
```

## 使い方

```bash
# 監視を開始
python main.py
```

起動すると `watch_target/` フォルダを監視します。

### 検知できるイベント

| イベント | 説明 |
|---------|------|
| 作成 | ファイル/フォルダが作成されたとき |
| 削除 | ファイル/フォルダが削除されたとき |
| 変更 | ファイルの内容が変更されたとき |

### 出力例

```
==================================================
🐱 Folder-Watching-Cat 起動！
   監視中: ./watch_target
   終了するには Ctrl+C を押してね
==================================================
🐱 にゃ！新しいファイルを見つけたよ！
   → ./watch_target/test.txt
```
